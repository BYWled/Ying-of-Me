莺系列整合包食用指南

下载整合包

在下载之前，你需要知道：整合包的命名极其统一以便于理解，因此选择一个符合自己需求的整合包极其重要。

1. 版本选择

按现在最新发布的Ying of Generation on 1.20.1（V2.0-B23.11.26-OCI）为例，下图为精简一图流：

· 整合包名称

图中的Ying of Generation就是莺纪元的名称(这不废话嘛)，除此之外，还有莺之诗Ying of Epic整合包。

· MC版本

图中的1.20.1是MC的版本号(这不还是废话嘛)，根据自己需要的MC版本下载就行。

请注意：由于模组对MC的版本兼容性问题，不要在整合包名称给出的版本以外的MC里加载整合包，那怕是小版本(1.20.1->1.20.2)也不行！

· 整合包版本

图中的V2.0是整合包版本号(我怀疑这是水贴)，符合MC版本的情况下建议越新越好，旧版本的整合包相比新版本可能存在模组过旧、bug过多、优化过差等等问题。

请注意：由于模组兼容性问题，在整合包版本不一致时：

  • 莺纪元：
    • （概率）存档损坏
    • （概率）游戏崩溃
    • （概率）无法加入多人世界
    • （部分模组）模组配置重置
  • 莺之诗
    • （必然）存档损坏
    • （概率）游戏崩溃
    • （概率）系统崩溃
    • （必然）无法加入多人世界
    • （部分模组）模组配置重置
· 测试模式

图中的B是测试模式，测试模式有以下几种：

  • A——Alpha：未发布的测试版本，仅提供内部测试使用
  • B——Beta：已经过测试的发布版本，基本没有影响游戏体验的bug，可以放心食用
  • D——Dev.：未测试版本，那么不推荐食用。如需使用请提前备份存档等游戏数据，避免bug导致数据损坏或丢失

所以，一般情况下，请寻找B模式 整合的包食用即可。

· 整合时间

图中23.11.26是整合时间，可得知该整合包的制作时间。(一般没什么用)如果使用的整合包制作时间过久，可能会出现模组版本过低，导致新物品&功能缺失的问题。(这个时候应该把作者踹去更新整合包)

· 整合方式

图中OCI是整合方式，整合方式有以下几种——

  • OCI——One Click Install：指包含了启动器的一键安装方式，该方式的内部包默认为F版
  • U——Ultra：指拓展整合方式，是基于额外追加了部分模组的整合包
  • F——Full：指完整整合方式，是功能完善的全量整合包
  • L——Lite：指精简整合方式，是舍去了部分功能，且额外加入了部分优化模组的整合包（该整合包不一定会在每个版本发布，且由于功能缺失，L版和同版本其他整合方式兼容性未知）

2.下载链接

· 频道下载

频道会根据需求上传所需MC版本的最新整合包，由于企鹅风控可能会上传失败，推荐前往网盘下载。加入频道点此。

· 网盘下载

网盘拥有所有的整合包资源：

  • 百度云——网盘界的老大哥，分享的文件100%失效…百度网盘，永久停更！
  • 阿里云——和楼上坐一桌，永久停更！
  • 蓝奏云——点此下载（密码：banying）：超过100Mb的文件需要分段解压，站内教程点此（由于维护工作量超标，大于100Mb的文件随缘更新）
  • 123网盘——点此下载（提取码：wled）

· 留言下载

如果上述链接都无法下载，你可以在评论区留下你的邮箱以及你需要的整合包版本，资源将随缘发送到你的邮箱。

安装方法

默认你已下载好了符合版本需求的Java

以下为不完全的版本需求对应的Java下载链接：

≤1.16.X的版本

Java 16

1.17.X-1.19.X

Java 17

1.20-1.20.4

Java 20

＞1.20.4的版本

Java 21

为了照顾早期无法使用高版本Java的电脑，此处提供Java 8的下载链接，但是高版本的MC几乎不支持Java 8运行。

1.OCI整合包

· 解压缩

下载完成后，你会得到一个Ying of XXX on X.XX.X(VX.X-XXX.XX.XX-OCI).zip文件，在你想要存放MC的位置，解压该文件内所有文件。

· 一键安装

解压完毕后，你会得到几个文件：

  • HMCL.exe
  • hmcl.json
  • modpack.zip

如果不是，请检查整合包是否为OCI整合模式！

双击HMCL.exe，并等待初始化、一键安装完毕。在安装完成后，modpack.zip可以删除。

请注意：OCI整合包可能存在整合时间过久导致一键无法安装的问题，如果无法自动安装也无需删除，请参考F版安装教程安装即可。

2.U&F&L 整合包

· 下载&准备启动器

U&F&L整合模式皆采用的MCBBS格式整合包整合，因此主流的启动器都可以识别整合包并安装。如需下载启动器，推荐如下：

  • HMCL——Hello Minecraft! Launcher：由huangyuhui 开发，具有六年历史、多系统支持、全新界面、自动安装、个性化主题、开源等特点
  • PCL2——Plain Craft Launcher 2：由龙腾猫跃 开发的一款我的世界启动器，它集成了众多游戏版本、Mod、整合包相关的内容，方便玩家一键下载和安装
  • BakaXL 4.0：一见惊鸿。这就是全新 BakaXL 启动器，采用了下一代技术构建。得益于全新的开发流程，现在的 BakaXL 不光体积轻盈，却仍然保留几乎所有强大的功能。

请注意：启动器存放的位置一般情况下就是你的游戏保存位置，请在合适位置放置启动器。

· 导入整合包

仅以HMCL启动器举例，其他启动器原理大致相同，自行安装。

一般的启动器可以直接把下载的整合包拖到启动器上就能安装

依次点击版本列表 -安装整合包 - 导入本地整合包文件



选择刚刚下载的整合包(如果OCI安装失败，此步选择modpack.zip)

导入整合包信息后确认安装版本无误点击安装 即可
