本内容提炼自前端核心基础课程笔记，首发伴莺の小窝，未经授权禁止搬运！

内容涵盖HTML、CSS、JS三大基础的高频面试题，采用“理解性记忆”法梳理。如有错漏，请向管理员反馈。

知识图谱

面试题汇总

一、HTML 与 浏览器基础篇

这部分主要考察你对页面骨架、HTML5 新特性以及浏览器存储、网络机制的理解。

H5 新特性与语义化

  • HTML5 引入了哪些新特性？
  • 常见的行内元素和块级元素有哪些？

标签与资源引入

  • link 和 @import 引入 CSS 的区别是什么？

本地存储（极高频）

  • localStorage、sessionStorage 和 cookie 的区别及各自的适用场景是什么？

同源策略与跨域

  • 浏览器控制 localStorage 访问的同源策略条件是什么？
  • 解决跨域有哪几种常见方式（如 JSONP 的理解及其优缺点、CORS）？

网络请求流程（经典题）

  • 从浏览器输入域名到服务端接收请求并渲染页面的中间完整流程是怎样的？（重点涉及 DNS 解析、TCP 握手、HTTP 请求等）。

HTTP 协议

  • HTTP 状态码中，服务端错误（5xx）和客户端错误（4xx）的典型值有哪些？
  • HTTP 与 TCP 的关系是什么？TCP 属于 OSI 哪一层？

二、CSS 布局与样式篇

CSS 的核心在于“布局（盒模型/Flex）”和“居中”。面试官通常会通过具体的场景来考察你的 CSS 功底。

盒模型（必考）

  • 说说你对 CSS 盒模型的理解？标准盒模型和怪异（IE）盒模型有什么区别？
  • 如何通过 box-sizing 更改盒模型？

元素居中（极高频）

  • 实现元素水平垂直居中的方法有哪些？（要求至少说出 Flex、绝对定位+Transform、Grid 等 3 种以上方案）。

Flex 弹性布局

  • Flex 布局的常见使用场景有哪些？
  • 主轴方向如何定义（如 flex-direction: row-reverse）？
  • 如何使用 Flex 实现居中？

CSS 单位与响应式

  • px、rem、em 单位的区别是什么？
  • rem 的相对根节点是什么？
  • 了解自适应和响应式布局的区别吗？

属性细节与继承

  • display 常用的值有哪些？
  • CSS 中哪些属性可以继承，哪些属性不可以继承？

Transform 细节

  • transform: translate(-50%, -50%) 中的 -50% 是针对谁的大小进行计算的？（答：自身宽高）。

CSS3 样式特性

  • CSS3 的新特性有哪些？
  • 直角边变圆角怎么实现？
  • 阴影（box-shadow）怎么实现？

三、JavaScript 核心与进阶篇

JS 是前端面试的重中之重，主要围绕数据类型、作用域/闭包、this 指向、异步编程（Promise）以及事件循环展开。

变量与数据类型

  • var、let、const 的区别是什么？
  • 什么情况下会出现 undefined？
  • 什么时候会出现 null？
  • typeof(null) 的输出结果是什么？（答："object"）。
  • JS 的数据类型有哪些？
  • 对 Symbol 和 BigInt 有什么了解？
  • 逻辑中断运算：当 1 || 2 执行时，左边为 true，右边还会执行吗？

作用域与闭包（高频痛点）

  • 什么是作用域链？
  • 谈谈你对闭包的理解？
  • 闭包的常见使用场景有哪些？（如在 React useState 或递归中的闭包）。

函数与 this 指向

  • 普通函数中 this 的指向规则是什么？
  • 如何改变普通函数 this 的指向？（掌握 call、apply、bind 的区别）。
  • 箭头函数的 this 指向规则是什么？
  • 能否通过 bind/call/apply 改变箭头函数的 this？

数组、对象操作与原生 JS

  • 常用数组遍历方式有哪些？（如 map、forEach、reduce 等）。
  • 实现数组浅拷贝和深拷贝的方法有哪些？（如扩展运算符 ...、concat、JSON.parse(JSON.stringify())、lodash.cloneDeep 等）。
  • 如何通过数组方法反转字符串？
  • 原生 JS 事件对象与 React 事件对象（合成事件）有何区别？

异步编程与事件循环（绝对核心）

  • 说说你对 JavaScript 事件循环（Event Loop）的理解？
  • Promise 异步的理解及常用方法？
  • new Promise 的参数结构是怎样的？
  • Promise 属于微任务还是宏任务？
  • 结合请求超时场景，如何手写 Promise 逻辑？
  • 使用 await 需要什么条件？（如必须在 async 函数内等）。

模块化与其他

  • CommonJS 和 AMD / ESModule 的区别是什么？
  • 如何处理后端返回的二进制文件（如 PDF、Blob 格式）？
